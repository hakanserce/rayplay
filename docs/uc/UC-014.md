# UC-014: Automatic Host Discovery

- **UC ID:** UC-014
- **Title:** Automatic Host Discovery
- **Priority:** P1
- **Estimated Complexity:** M

## Description

As a client user, I want my client to automatically discover available RayHost servers
on the local network so that I do not need to manually type IP addresses.

## Acceptance Criteria

1. RayHost advertises its presence on the local network while running
2. RayView discovers available hosts and displays them in a list
3. Discovery works without any manual configuration on either side
4. The client shows the host's display name, address, and current status
5. Discovery completes within 3 seconds on a typical home network
6. The user can still connect manually by IP address if discovery fails

## Technical Approach

Use a LAN service discovery mechanism to advertise and find hosts. The CLI can
list discovered hosts or the user can specify an address directly.
