# UC-019: Multi-Monitor Selection

- **UC ID:** UC-019
- **Title:** Multi-Monitor Selection
- **Priority:** P2
- **Estimated Complexity:** M

## Description

As a host user, I want to select which monitor to stream when I have multiple displays
so that I can control what the client sees.

## Acceptance Criteria

1. RayHost detects all connected monitors and lists them
2. The user can select a specific monitor to stream via CLI option
3. The default is the primary monitor if no selection is made
4. Monitor selection can be changed without restarting the server
5. Each monitor's resolution and position are reported correctly

## Technical Approach

Extend the screen capture from UC-001 to enumerate all displays and capture from
the selected one.

## Dependencies

- UC-001 (Host Screen Capture)
