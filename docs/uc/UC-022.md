# UC-022: Android Client

- **UC ID:** UC-022
- **Title:** Android Client
- **Priority:** P1
- **Estimated Complexity:** L

## Description

As a user, I want to stream games to an Android device so that I can play my PC
games on a tablet or phone.

## Acceptance Criteria

1. An Android app connects to a RayHost server and displays the video stream
2. Touch input on the Android device is mapped to mouse movement and clicks
3. On-screen virtual controls provide keyboard shortcuts for common gaming actions
4. The stream runs at a stable frame rate appropriate for the device
5. The app supports both landscape and portrait orientations
6. An external gamepad connected to the Android device works (via UC-020)

## Technical Approach

Port the client-side pipeline (decoding, rendering, input capture) to Android.
Reuse the core Rust logic via cross-compilation and platform-specific rendering/input
layers.

## Dependencies

- UC-008 (End-to-End Video Streaming)
- UC-020 (Gamepad Input Support)
