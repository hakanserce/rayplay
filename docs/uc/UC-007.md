# UC-007: Client CLI (RayView)

- **UC ID:** UC-007
- **Title:** Client CLI (RayView)
- **Priority:** P0
- **Estimated Complexity:** S

## Description

As a client user, I want to connect to a streaming host from the command line by
specifying its address so that I can view the host's screen.

## Acceptance Criteria

1. Running `rayview <host-address>` connects to the host and opens a rendering window
2. The CLI accepts configuration options: host address, port, window size
3. Connection errors are reported clearly (host unreachable, refused, timeout)
4. The client disconnects cleanly on window close or Ctrl+C
5. Sensible defaults are provided for all configuration options

## Technical Approach

Build a CLI application using the workspace's CLI crate that orchestrates the
transport ([UC-003](UC-003.md)), decoding ([UC-004](UC-004.md)), and rendering ([UC-005](UC-005.md)) pipeline.
