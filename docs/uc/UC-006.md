# UC-006: Host CLI (RayHost)

- **UC ID:** UC-006
- **Title:** Host CLI (RayHost)
- **Priority:** P0
- **Estimated Complexity:** S

## Description

As a host user, I want to start a streaming server from the command line so that
I can begin hosting a session for a remote client to connect to.

## Acceptance Criteria

1. Running `rayhost` starts the server and begins listening for client connections
2. The CLI accepts configuration options: port, resolution, frame rate, bitrate
3. The server logs its listening address and port on startup
4. The server can be stopped cleanly with Ctrl+C
5. Sensible defaults are provided for all configuration options

## Technical Approach

Build a CLI application using the workspace's CLI crate that orchestrates the
capture (UC-001), encoding (UC-002), and transport (UC-003) pipeline.

## Dependencies

- UC-002 (Host Video Encoding)
- UC-003 (Video Stream Transport)
